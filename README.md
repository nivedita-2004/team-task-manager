# Team Task Manager (Full-Stack)

A modern, responsive Team Task Manager application built with a React frontend (Vite), Node.js backend (Express), and a MySQL database (using raw `mysql2` queries).

## Key Features
- **Authentication**: JWT-based secure signup and login.
- **Role-based Access**: `ADMIN` and `MEMBER` roles.
- **Project Management**: Create and view projects. Admins can create new ones.
- **Task Management**: Create tasks, assign them to members via dynamic dropdowns, and update statuses.
- **Dashboard**: Split Dashboards! Admins get a "Command Center" overview, while Members get a "My Workspace" focused on their tasks.
- **Premium Design**: Custom Vanilla CSS featuring "Blush & White" glassmorphism, responsive grids, and modern typography. Fully mobile responsive.

##  Tech Stack
- **Frontend**: React (Vite), React Router, Axios, Lucide React, Vanilla CSS.
- **Backend**: Node.js, Express, mysql2, JSONWebToken, Bcryptjs.
- **Database**: MySQL.
